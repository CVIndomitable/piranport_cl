# 皮兰港 Mod 配置说明

配置文件由 NeoForge 自动生成，位于 Minecraft 实例的 `config/` 目录下。

---

## 通用配置（服务端+客户端共享）

**文件路径：** `config/piranport-common.toml`

### `shipCoreGuiEnabled`

| 项目 | 内容 |
|------|------|
| 类型 | Boolean |
| 默认值 | `false` |
| 说明 | 是否启用舰装核心 GUI 界面。 |

- **`false`（默认，无 GUI 模式）：**
  - 右键舰装核心只触发变身/发射，不打开背包界面。
  - 武器直接放在玩家背包里，装甲板也直接放背包，无需放入核心容器槽。
  - **发射逻辑：主手持核心→副手武器发射；副手持核心→主手武器发射。** 切换武器只需交换手持物品，无需按 V 键。
  - 负重由背包内所有武器/装甲的重量总和决定，以背包中最高负重上限的核心为基准。

- **`true`（GUI 模式）：**
  - 未变身时右键核心打开舰装核心界面，可将武器拖入武器槽、弹药拖入弹药舱、装甲放入强化槽。
  - 变身后右键发射当前选中武器，按 **V** 键循环切换武器槽。
  - 负重和弹药仅读取核心容器槽内的物品。

---

### `fighterAmmoEnabled`

| 项目 | 内容 |
|------|------|
| 类型 | Boolean |
| 默认值 | `false` |
| 说明 | 是否启用战斗机（战斗机编队）的子弹消耗。 |

- **`false`（默认）：** 战斗机子弹无限，仅消耗燃料。每次出击前通过变身自动补充燃料。
- **`true`：** 战斗机每次攻击消耗 1 发子弹（`ammoCapacity` 递减），弹尽后返航。子弹由编组界面配置的弹种提供。

---

### `autoResupplyEnabled`

| 项目 | 内容 |
|------|------|
| 类型 | Boolean |
| 默认值 | `false` |
| 说明 | 控制所有武器（舰炮、鱼雷、飞机）的装填方式。 |

- **`false`（默认，手动装填模式）：**
  - **舰炮**：需要手动装弹——在背包界面，将鼠标拖起弹药，右键点击炮的物品格，每次装入 1 发。炮只有装弹完毕且冷却结束才显示绿色"已装填"。
  - **鱼雷发射器**：需要逐管装填——右键鱼雷装入 1 枚（需重复操作直到装满所有管）。如四联鱼雷需右键 4 次才算装填就绪。
  - **飞机**：需要手动加油——右键 `航空燃料` 到飞机物品格完成燃料填充。
  - 变身时 **不会** 自动消耗背包弹药或燃料。

- **`true`（自动装填模式）：**
  - 发射时自动从背包中查找并消耗对应弹药，变身时自动从弹药槽填充飞机燃料。
  - 无需手动操作，只要背包有弹药即可发射。

---

### `flammableEffectEnabled`

| 项目 | 内容 |
|------|------|
| 类型 | Boolean |
| 默认值 | `false` |
| 说明 | 是否在飞机装燃料后给玩家施加「易燃易爆」Debuff。 |

- **`false`（默认）：** 飞机装载燃料不附加任何状态效果，无惩罚。
- **`true`：** 变身后携带已加燃料的飞机时（或手动给飞机加油后变身状态中），玩家获得「易燃易爆」效果——受伤时有概率触发小型爆炸，增加危险性。

## 客户端配置（仅本地）

**文件路径：** `config/piranport-client.toml`

### `showLegacyReloadHud`

| 项目 | 内容 |
|------|------|
| 类型 | Boolean |
| 默认值 | `false` |
| 说明 | 是否显示旧版装填进度 HUD 条（热键栏上方横条）。 |

- **`false`（默认）：** 装填进度通过舰装核心物品图标上的装饰器（物品右下角小图标）显示。
- **`true`：** 恢复显示旧版热键栏上方的横条进度条。

---

### `flightGroupEnabled`

| 项目 | 内容 |
|------|------|
| 类型 | Boolean |
| 默认值 | `false` |
| 说明 | 是否在舰装核心 GUI 中显示「编组」按钮（飞机编队配置界面）。 |

- **`false`（默认）：** 编组按钮隐藏。可通过 **U 键** 在变身后直接打开编组界面。
- **`true`：** 在舰装核心 GUI 底部显示「编组」按钮。
