# 皮兰港 (Piran Port) Mod

> 基于战舰少女R世界观的 Minecraft 综合性模组

## 安装

1. 安装 **NeoForge 21.1.x**（对应 Minecraft 1.21.1）
2. 将 `piranport-0.0.11.jar` 放入 `mods/` 文件夹
3. （可选）安装 JEI 以查看自定义配方
4. 启动游戏

---

## 两种操作模式

模组支持两种模式，通过 `config/piranport-common.toml` 中的 `shipCoreGuiEnabled` 切换。

### 无 GUI 模式（默认，`shipCoreGuiEnabled = false`）

```
1. 挖矾土矿 → 熔炼铝锭，收集盐
2. 合成舰装核心，放入背包（无需右键打开GUI）
3. 武器/装甲直接放背包，核心自动识别负重
4. Shift+右键核心 → 变身启动
5. 主手持武器/飞机 → 右键发射/放飞
6. P键锁定目标，放飞飞机后飞机自动攻击锁定目标
7. 侦察机放飞后自动切换视角，WASD操控，V键退出
```

### GUI 模式（`shipCoreGuiEnabled = true`）

```
1. 右键舰装核心 → 打开配装界面
2. 武器槽放武器，弹药库放弹药，强化槽放装甲
3. U键编组飞机（需先变身）
4. Shift+右键 → 变身，右键发射，V键切换武器槽
```

## 按键一览

| 按键 | 功能（无GUI模式） | 功能（GUI模式） |
|------|-----------------|----------------|
| `Shift + 右键` | 变身启动/解除 | 变身启动/解除 |
| `右键` | 主手武器发射/飞机放飞 | 发射当前槽武器/飞机放飞 |
| `V` | 退出侦察机视角 | 退出侦察机 / 切换武器槽 |
| `U` | 打开编组 GUI（变身中） | 打开编组 GUI（变身中） |
| `P` | 火控锁定准星目标（变身中）| 同左 |
| `O` | 火控加选目标 | 同左 |
| `I` | 取消所有火控锁定 | 同左 |
| `Y` | 切换战场高亮（自机/锁定目标） | 同左 |
| `F8` | 切换调试模式 / Shift+F8 输出快照 | 同左 |

## 舰装核心参数

| 核心 | 武器槽 | 弹药槽 | 强化槽 | 负重上限 |
|------|--------|--------|--------|---------|
| 小型 | 4 | 4 | 2 | 40 |
| 中型 | 5 | 4 | 3 | 64 |
| 大型 | 6 | 4 | 4 | 112 |

移速倍率 = 1.0 − (负重/上限) × 0.6，最低 40%。

无GUI模式下以背包中**负重上限最高**的核心为基准。

## 副本系统

变身状态下，与副本讲台交互进入副本选关界面。

| 要素 | 说明 |
|------|------|
| 副本讲台 | 放置后右键打开副本选关界面 |
| 副本钥匙 | 消耗品，用于开启副本实例 |
| 节点战斗 | 每个副本由多个节点组成，逐节点推进 |
| 箱子船 | 击败节点后出现，右键领取战利品 |
| 传送门 | 节点间传送，自动引导进度 |
| 回城卷轴 | 使用后确认返回主世界（消耗品） |
| 复活机制 | 副本内死亡可选择复活或退出 |
| 排行榜 | 记录副本通关成绩 |

## 遗迹系统

主世界海洋中自然生成四类遗迹结构，包含深海NPC守卫和战利品箱。

| 遗迹类型 | 说明 | 间距参考 |
|----------|------|---------|
| 传送门遗迹 | 含深海传送门框架 | 40 chunks |
| 补给站 | 轻型补给点 | 32 chunks |
| 前哨站 | 中型据点 | 48 chunks |
| 深海基地 | 大型基地，高级战利品 | 64 chunks |

遗迹内的一次性刷怪方块（AbyssalSpawner）在结构首次加载时生成深海NPC后自毁。战利品由 `LootChestProcessor` 注入对应等级的战利品表。

## 深海NPC

9种深海敌人，数据驱动（`data/piranport/npc/deep_ocean/` JSON 配置），支持集群AI。

| NPC | 说明 |
|-----|------|
| 补给酱 | 基础补给单位 |
| 深海驱逐 | 轻型战斗 |
| 深海轻巡 | 轻巡洋舰 |
| 深海重巡 | 重巡洋舰 |
| 深海战巡 | 战斗巡洋舰 |
| 深海战列 | 战列舰，高血量高火力 |
| 深海轻母 | 轻型航母 |
| 深海航母 | 正规航母，可放飞飞机 |
| 深海潜艇 | 潜艇，可下潜 |

**弹道系统**：深海弹丸支持抛物线/追踪/直射三种模式，含提前量计算和近炸引信。

**集群AI**：同舰队组（clusterUUID）的敌人共享首个发现的目标，已交战者不切换目标。

## 调试命令

变身状态下可使用 `/ppd` 调试命令（需OP权限）：

| 命令 | 说明 |
|------|------|
| `/ppd spawn_ruin <type>` | 强制在当前位置生成指定类型遗迹 |
| `/ppd spawn_abyssal <type> [count]` | 刷出指定类型深海NPC |
| `/ppd locate_ruin <type>` | 定位最近的指定类型遗迹 |

## 火控系统

变身状态下，无论 GUI 模式还是无 GUI 模式，**火控 HUD 均正常显示**（右上角目标面板 + 血条）。

| 操作 | 效果 |
|------|------|
| P键对准目标 | 单锁（替换旧目标），最远80格 |
| O键对准目标 | 追加锁定，最多4个 |
| I键 | 取消全部锁定 |
| FOCUS编组 | 全组攻击锁定列表第一个目标 |
| SPREAD编组 | 各飞机攻打最近的锁定目标 |
| 无锁定 | 自动索敌32格内最近敌对生物 |

## 配置速查

**通用配置** (`config/piranport-common.toml`)：

| 配置项 | 默认 | 说明 |
|--------|------|------|
| `shipCoreGuiEnabled` | `false` | `true` 启用GUI配装模式 |
| `autoResupplyEnabled` | `false` | `true` 切换为自动装填（变身时自动扣弹药） |
| `fighterAmmoEnabled` | `false` | `true` 战斗机消耗子弹 |
| `flammableEffectEnabled` | `false` | `true` 开启飞机燃料易燃易爆惩罚 |
| `saltGenerationEnabled` | `false` | `true` 启用河流盐矿自然生成 |
| `weaponPickupToInventory` | `false` | `true` 武器拾取直接进背包 |
| `ruinGenerationEnabled` | `true` | 主世界遗迹生成总开关 |
| `abyssalEnemyDifficultyMultiplier` | `1.0` | 深海NPC属性倍率 |

**客户端配置** (`config/piranport-client.toml`)：

| 配置项 | 默认 | 说明 |
|--------|------|------|
| `showLegacyReloadHud` | `false` | `true` 显示旧版装填横条HUD |
| `flightGroupEnabled` | — | 编组GUI开关 |

完整配置说明见 [CONFIG.md](CONFIG.md)
