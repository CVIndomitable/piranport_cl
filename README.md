# 皮兰港 (Piran Port) Mod

> 基于战舰少女R世界观的 Minecraft 综合性模组
> NeoForge 21.1.220 for Minecraft 1.21.1 | 版本 **v0.0.4-alpha**
> 由 Claude Code (AI) 协助开发

---

## 功能概览

| 系统 | 内容 |
|------|------|
| 🚢 **舰装系统** | 舰装核心 GUI、变身、负重/移速、武器切换 |
| 💥 **火炮战斗** | 小/中/大型火炮 + HE/AP 炮弹 |
| 🐟 **鱼雷系统** | 三种发射器、进水Debuff、扇形散布 |
| ✈️ **航空系统** | 4种飞机编队、编组GUI、火控P/O/I、战场高亮Y |
| 🍳 **厨房系统** | 7种作物、石磨/砧板/厨锅、15+种食物+Buff |

---

## 安装

1. 安装 **NeoForge 21.1.x**（对应 Minecraft 1.21.1）
2. 将 `piranport-0.0.4.jar` 放入 `mods/` 文件夹
3. 启动游戏

---

## 快速上手

```
1. 挖矾土矿 → 熔炼铝锭，收集盐
2. 合成舰装核心（小/中/大型），右键打开 GUI
3. 武器栏放火炮/鱼雷发射器，弹药库放对应弹药
4. Shift+右键 变身启动，右键发射，V键切换武器
5. 放入飞机编队+航空燃料+弹药，U键打开编组配置后放飞
```

---

## 按键一览

| 按键 | 功能 |
|------|------|
| `Shift + 右键` | 变身启动/解除 |
| `右键` | 发射当前武器 / 放飞下一编组 |
| `V` | 切换武器槽 |
| `U` | 打开编组 GUI（变身中） |
| `P` | 火控锁定准星目标 |
| `O` | 火控加选目标 |
| `I` | 取消所有火控锁定 |
| `Y` | 切换战场高亮 |

## 舰装核心参数

| 核心 | 武器槽 | 弹药槽 | 强化槽 | 负重上限 |
|------|--------|--------|--------|---------|
| 小型 | 2 | 4 | 1 | 40 |
| 中型 | 4 | 6 | 2 | 72 |
| 大型 | 6 | 8 | 3 | 112 |

移速倍率 = 1.0 − (负重/上限) × 0.6，最低 40%。
