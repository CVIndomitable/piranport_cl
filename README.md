# 皮兰港 (Piran Port) Mod

> 基于战舰少女R世界观的 Minecraft 综合性模组
> NeoForge 21.1.220 for Minecraft 1.21.1 | 版本 **v0.0.7-alpha**
> 由 Claude Code (AI) 协助开发

---

## 功能概览

| 系统 | 内容 |
|------|------|
| 🚢 **舰装系统** | 舰装核心、变身、负重/移速、无GUI模式（默认）/ GUI配装模式 |
| 💥 **火炮战斗** | 小/中/大型火炮 + HE/AP 炮弹，手动装填或自动装填 |
| 🐟 **鱼雷系统** | 双/三/四联发射器、进水Debuff、扇形散布 |
| ✈️ **航空系统** | 5种飞机编队、编组GUI、火控P/O/I（两种模式均显示HUD）、空战、编队跟随侦察机、自动升空 |
| 🍳 **厨房系统** | 7种作物、石磨/砧板/厨锅（支持漏斗自动化）、15+种食物+Buff |
| ⚡ **战斗Buff** | 装填加速（×2/×3）、高速规避（15%/25%/35%） |

---

## 安装

1. 安装 **NeoForge 21.1.x**（对应 Minecraft 1.21.1）
2. 将 `piranport-0.0.7.jar` 放入 `mods/` 文件夹
3. 启动游戏

---

## 两种操作模式

模组支持两种模式，通过 `config/piranport-common.toml` 中的 `shipCoreGuiEnabled` 切换。

### 无 GUI 模式（默认，`shipCoreGuiEnabled = false`）

```
1. 挖矾土矿 → 熔炼铝锭，收集盐
2. 合成舰装核心，放入背包（无需右键打开GUI）
3. 武器/装甲直接放背包，核心自动识别负重
4. Shift+右键核心 → 变身启动
5. 主手持武器/飞机 → 右键发射/放飞
6. P键锁定目标，放飞飞机后飞机自动攻击锁定目标
7. 侦察机放飞后自动切换视角，WASD操控，V键退出
```

### GUI 模式（`shipCoreGuiEnabled = true`）

```
1. 右键舰装核心 → 打开配装界面
2. 武器槽放武器，弹药库放弹药，强化槽放装甲
3. U键编组飞机（需先变身）
4. Shift+右键 → 变身，右键发射，V键切换武器槽
```

## 按键一览

| 按键 | 功能（无GUI模式） | 功能（GUI模式） |
|------|-----------------|----------------|
| `Shift + 右键` | 变身启动/解除 | 变身启动/解除 |
| `右键` | 主手武器发射/飞机放飞 | 发射当前槽武器/飞机放飞 |
| `V` | 退出侦察机视角 | 退出侦察机 / 切换武器槽 |
| `U` | 打开编组 GUI（变身中） | 打开编组 GUI（变身中） |
| `P` | 火控锁定准星目标（变身中）| 同左 |
| `O` | 火控加选目标 | 同左 |
| `I` | 取消所有火控锁定 | 同左 |
| `Y` | 切换战场高亮（自机/锁定目标） | 同左 |
| `F8` | 切换调试模式 / Shift+F8 输出快照 | 同左 |

## 舰装核心参数

| 核心 | 武器槽 | 弹药槽 | 强化槽 | 负重上限 |
|------|--------|--------|--------|---------|
| 小型 | 4 | 4 | 2 | 40 |
| 中型 | 5 | 4 | 3 | 64 |
| 大型 | 6 | 4 | 4 | 112 |

移速倍率 = 1.0 − (负重/上限) × 0.6，最低 40%。

无GUI模式下以背包中**负重上限最高**的核心为基准。

| 飞机 | 负重 | 攻击方式 | 贴图 |
|------|------|---------|------|

## 火控系统

变身状态下，无论 GUI 模式还是无 GUI 模式，**火控 HUD 均正常显示**（右上角目标面板 + 血条）。

| 操作 | 效果 |
|------|------|
| P键对准目标 | 单锁（替换旧目标），最远80格 |
| O键对准目标 | 追加锁定，最多4个 |
| I键 | 取消全部锁定 |
| FOCUS编组 | 全组攻击锁定列表第一个目标 |
| SPREAD编组 | 各飞机攻打最近的锁定目标 |
| 无锁定 | 自动索敌32格内最近敌对生物 |

## 配置速查

| 配置项 | 默认 | 说明 |
|--------|------|------|
| `shipCoreGuiEnabled` | `false` | `true` 启用GUI配装模式 |
| `autoResupplyEnabled` | `false` | `true` 切换为自动装填（变身时自动扣弹药） |
| `fighterAmmoEnabled` | `false` | `true` 战斗机消耗子弹 |
| `flammableEffectEnabled` | `false` | `true` 开启飞机燃料易燃易爆惩罚 |
| `showLegacyReloadHud` | `false` | `true` 显示旧版装填横条HUD |

完整配置说明见 [CONFIG.md](CONFIG.md)
