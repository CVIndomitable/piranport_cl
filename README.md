# 皮兰港 (Piran Port) Mod

> 基于战舰少女R世界观的 Minecraft 综合性模组
> NeoForge 21.1.220 for Minecraft 1.21.1 | 版本 **v0.0.7-alpha**
> 由 Claude Code (AI) 协助开发

---

## 功能概览

| 系统 | 内容 |
|------|------|
| 🚢 **舰装系统** | 舰装核心 GUI、变身、负重/移速、武器切换、舰装配置 |
| 💥 **火炮战斗** | 小/中/大型火炮 + HE/AP 炮弹 |
| 🐟 **鱼雷系统** | 三种发射器、进水Debuff、扇形散布 |
| ✈️ **航空系统** | 5种飞机（含侦察机）、编组GUI、火控P/O/I、空战、编队跟随、自动升空 |
| 🍳 **厨房系统** | 7种作物、石磨/砧板/厨锅（支持漏斗自动化）、15+种食物+Buff |
| ⚡ **战斗Buff** | 装填加速（×2/×3）、高速规避（15%/25%/35%） |

---

## 安装

1. 安装 **NeoForge 21.1.x**（对应 Minecraft 1.21.1）
2. 将 `piranport-0.0.7.jar` 放入 `mods/` 文件夹
3. 启动游戏

---

## 快速上手

```
1. 挖矾土矿 → 熔炼铝锭，收集盐
2. 合成舰装核心（小/中/大型），右键打开 GUI
3. 武器栏放火炮/鱼雷发射器，弹药库放对应弹药
4. Shift+右键 变身启动，右键发射，V键切换武器
5. 放入飞机编队+航空燃料+弹药，U键打开编组配置后放飞
6. 侦察机编队：放飞后自动切换视角，WASD操控飞行
7. 舰装配置：开启"战斗机自动升空"可自动对幻翼升空迎击
```

---

## 按键一览

| 按键 | 功能 |
|------|------|
| `Shift + 右键` | 变身启动/解除 |
| `右键` | 发射当前武器 / 放飞下一编组 |
| `V` | 切换武器槽 |
| `U` | 打开编组 GUI（变身中） |
| `P` | 火控锁定准星目标 |
| `O` | 火控加选目标 |
| `I` | 取消所有火控锁定 |
| `Y` | 切换战场高亮 |
| `G` | 退出侦察机视角 |

## 舰装核心参数

| 核心 | 武器槽 | 弹药槽 | 强化槽 | 负重上限 |
|------|--------|--------|--------|---------|
| 小型 | 4 | 4 | 2 | 40 |
| 中型 | 5 | 4 | 3 | 64 |
| 大型 | 6 | 4 | 4 | 112 |

移速倍率 = 1.0 − (负重/上限) × 0.6，最低 40%。

---

## 飞机编队参数

| 飞机 | 负重 | 弹药 | 攻击方式 |
|------|------|------|---------|
| 战斗机编队 | 8 | 子弹（64发） | 悬停射击 |
| 俯冲轰炸机编队 | 10 | 航空炸弹（1枚/次） | 俯冲接触投弹 |
| 鱼雷轰炸机编队 | 12 | 航空鱼雷（1枚/次） | 低飞投雷 |
| 水平轰炸机编队 | 12 | 航空炸弹（8枚） | 高空水平投弹 |
| 侦察机编队 | 8 | 航空燃料 | 视角切换/WASD操控 |
