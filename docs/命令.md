# 皮兰港 命令参考

本文档列出皮兰港模组新增的所有命令和调试快捷键。

---

## 聊天命令

所有命令以 `/ppd`（Piran Port Debug）为前缀，需要 **OP 权限等级 2** 以上。

### `/ppd spawn_ruin <type>`

在玩家当前位置生成一个占位遗迹结构。

| 参数 | 可选值 | 说明 |
|------|--------|------|
| `type` | `portal_ruin` | 传送门遗迹（8格半径） |
| | `supply_depot` | 补给站（12格半径） |
| | `outpost` | 前哨站（24格半径） |
| | `abyssal_base` | 深海基地（40格半径） |

会生成石砖地板、苔石砖围墙和一个战利品箱子。`portal_ruin` 类型额外生成深渊传送门框架。

**示例：**
```
/ppd spawn_ruin portal_ruin
/ppd spawn_ruin abyssal_base
```

---

### `/ppd spawn_abyssal <entity_type> [count]`

在玩家周围生成深海 NPC 实体。

| 参数 | 说明 |
|------|------|
| `entity_type` | 深海实体 ID（见下表） |
| `count` | 生成数量，1-20，默认为 1 |

**可用实体类型：**

| entity_type | 舰种 |
|-------------|------|
| `deep_ocean_supply` | 补给酱 |
| `deep_ocean_destroyer` | 深海驱逐 |
| `deep_ocean_light_cruiser` | 深海轻巡 |
| `deep_ocean_heavy_cruiser` | 深海重巡 |
| `deep_ocean_battle_cruiser` | 深海战巡 |
| `deep_ocean_battleship` | 深海战列 |
| `deep_ocean_light_carrier` | 深海轻母 |
| `deep_ocean_carrier` | 深海航母 |
| `deep_ocean_submarine` | 深海潜艇 |

生成的实体会在玩家周围 6 格范围内随机分布。

**示例：**
```
/ppd spawn_abyssal deep_ocean_destroyer
/ppd spawn_abyssal deep_ocean_battleship 5
```

---

### `/ppd locate_ruin <type>`

提示使用原版 `/locate` 命令定位指定类型的遗迹结构。

| 参数 | 可选值 |
|------|--------|
| `type` | `portal_ruin` / `supply_depot` / `outpost` / `abyssal_base` |

实际定位请使用：
```
/locate structure piranport:<type>
```

## 调试快捷键

以下快捷键通过 `F8` 键触发，不需要聊天窗口。

### F8 — 调试模式开关

切换调试模式的开启/关闭。开启后：
- 屏幕显示调试水印
- 服务端开始记录性能埋点和事件日志
- 日志输出到 `logs/piranport-debug.log`

聊天栏会显示 `[PP DEBUG] ON` 或 `[PP DEBUG] OFF`。

### Shift + F8 — 状态快照

将当前玩家的完整状态快照写入 `logs/piranport-debug.log`，包含：
- 变身状态（核心类型、负重百分比、移动速度）
- 当前生效的 Buff 列表
- 火控锁定目标列表
- 活跃飞机列表（类型、状态）

无论调试模式是否开启，快照功能始终可用。
