# 皮兰港模组教程

本教程面向玩家，介绍皮兰港模组的核心玩法。

---

## 模组功能概览

### 1. 舰装系统

#### 无 GUI 模式（默认）

将**舰装核心**（小/中/大型）放入**副手槽**，自动进入变身状态；从副手取走，自动解除变身（同时召回所有飞机）。无需 Shift+右键。

- **武器**：直接放在背包/快捷栏中，副手持核心时主手持武器即可发射
- **装甲板**：必须存入核心才生效（见下方"装甲存放"说明）
- **负重**：背包内武器重量 + 核心内装甲重量之和，以副手核心负重上限为基准
- **移速**：负重越接近上限，移速越慢（最低约40%）

#### 装甲存放（无 GUI 模式）

装甲板放在背包里**不生效**，需存入核心：

1. 按 **E** 打开背包
2. 用鼠标拿起装甲板（右键 = 拿起1个）
3. 右键点击副手槽中的舰装核心 → 装甲板存入（最多2–4槽，取决于核心型号）
4. 空手右键副手核心格 → 取出最后一块装甲

悬停查看核心物品可看到当前储存护甲信息和护甲值。

#### GUI 模式（需开启 `shipCoreGuiEnabled = true`）

Shift+右键手动切换变身状态，右键打开舰装栏 GUI：
- **武器栏**（左侧）：装备火炮、鱼雷发射器或飞机编队
- **弹药库**（中间）：存放炮弹/鱼雷/航空弹药/航空燃料
- **强化栏**（右侧）：装备附加装甲（+2/+4/+6防御）

#### 武器切换（GUI 模式）

`V`键切换当前选中武器槽，右键发射/放飞当前武器。

#### 装填进度显示

变身后，**快捷栏中舰装核心图标**上会显示彩色装填进度条（每个武器槽一条）：

| 颜色 | 武器类型 | 说明 |
|------|---------|------|
| 橙红色 | 火炮 | 亮橙 = 可发射 |
| 蓝色 | 鱼雷发射器 | 亮蓝 = 可发射 |
| 绿色 | 飞机编队 | 亮绿 = 可放飞 |

条从左向右填满表示装填完成，变亮即可再次发射。

每个武器槽**独立冷却**，不同武器互不影响——装填中的大炮不会阻止你发射鱼雷或放飞飞机。

#### 舰装配置

舰装栏 GUI 底部设有**舰装配置**区域，包含可开关的自动化选项（详见第9节）。

### 2. 火炮战斗

| 武器 | 联装数 | 弹药 | 特性 |
|------|--------|------|------|
| 单装小型火炮 | 1门 | 小口径 HE/AP | 装填快，单发，弹药消耗少 |
| 小型火炮 | 2门 | 小口径 HE/AP | 装填快，双联齐射 |
| 中型火炮 | 1门 | 中口径 HE/AP | 均衡 |
| 大型火炮 | 3门 | 大口径 HE/AP | 三联齐射，伤害高，装填慢 |

> **联装数**：每次发射消耗对应数量的炮弹（如大型火炮一次齐射消耗3发）。装填时也需要一次性放入足够数量的炮弹。

- **HE炮弹**：爆炸伤害，范围溅射
- **AP炮弹**：穿甲，对高护甲目标更有效
- **三式弹**：对空特化弹种（小/中/大口径均有）

#### 手动装填（R键）

**大型和中型火炮**支持 R 键手动装填：

1. 变身状态下，主手持大型或中型火炮
2. 按 **R 键**，系统自动从背包中寻找匹配口径的弹药装入
3. 弹药选择逻辑与自动装弹一致
4. 小型火炮不支持手动装填（始终自动装填）

> 已装弹的武器按 R 键会提示"已装弹"。

### 4. 导弹系统

导弹分为三类：**反舰导弹**（追踪水面/地面目标）、**防空导弹**（追踪空中目标）、**火箭弹**（直射无制导）。

#### 导弹发射器

| 发射器 | 导弹类型 | 弹药 | 特性 |
|--------|---------|------|------|
| 上游一号发射器 | 反舰 | 上游一号 | 伤害30+6穿甲，连装2，负重25 |
| MK14鱼叉发射器 | 反舰 | 鱼叉导弹 | 伤害24，连装4，负重22 |
| 小猎犬发射器 | 防空 | 小猎犬导弹 | 伤害9，冷却60s，负重14 |
| 箭型防空导弹 | 防空 | 防空导弹 | 伤害6，冷却60s，负重7 |
| 海猫防空导弹 | 防空 | 防空导弹 | 伤害6，冷却60s，负重6 |
| 舰载火箭弹 | 火箭 | 火箭弹 | 直射，无制导 |

#### 导弹使用

- 反舰导弹：右键发射，自动追踪前方最近目标（排除水下目标和容器实体）；也可通过**火控系统**锁定目标后发射，导弹将追踪锁定目标
- 防空导弹：开启**自动升空**（H键）后，32格内有**空中敌对生物**时自动发射拦截；也可手动右键发射。自动发射时优先攻击火控锁定目标，无锁定时搜索最近空中敌人
- 火箭弹：直线飞行，无制导

> 导弹武器冷却条为独立的**黄色**进度条，区别于火炮的红色和鱼雷的蓝色。
> 
> **未装填提示**：导弹发射器和鱼雷发射器未装填弹药时，物品图标上会显示空的耐久条，提醒玩家需要装填。大中型火炮同理。

### 5. 深弹系统

深水炸弹用于攻击水下目标（潜艇、水生生物）。

| 投射器 | 投掷数 | 散布模式 |
|--------|--------|---------|
| 深弹投射器 | 1发 | 单发直投 |
| 深弹投射器·改 | 2发 | 前后散布 |
| 深弹投射器·改二 | 3发 | 三角散布 |

- 弹药：**深水炸弹**
- 深弹投入水中后自动下沉，接近目标时**近炸引信**触发爆炸
- 造成**区域魔法伤害**，对周围水下实体均有效
- 深弹为独立的 `DEPTH_CHARGE` 武器分类

### 7. 武器与弹药合成

#### 武器合成台

放置后右键打开 GUI，包含5个标签页：

| 标签页 | 可合成内容 |
|--------|-----------|
| 火炮 | 各型号火炮 |
| 鱼雷发射器 | 各联装发射器 |
| 导弹发射器 | 各型号导弹发射器 |
| 深弹 | 各型号深弹投射器 |
| 飞机 | 各型号飞机编队 |

- 每个标签页有下拉选择器，选择具体要合成的武器
- 部分高级武器需要**蓝图**（如中型火炮蓝图、大型火炮蓝图）
- 选择配方后，放入所需材料，点击"合成"开始定时合成
- 合成进度显示在 GUI 中，完成后取出成品

#### 弹药合成台

放置后右键打开 GUI，可合成各种弹药（炮弹、鱼雷弹头、导弹、深水炸弹等）。

#### 装填设施

放置后右键打开 GUI，支持将弹药装填到发射器中：

- 将**空的发射器**（火炮/鱼雷发射器/导弹发射器）放入左侧槽位
- 将**对应弹药**放入右侧槽位
- 自动装填完成后取出已装弹的发射器

### 9. 航空系统

#### 9.1 飞机编队物品

六种飞机编队放入武器栏，变身后通过编组配置后右键放飞：

| 飞机 | 弹药 | 负重 | 攻击方式 |
|------|------|------|---------|
| 战斗机编队 | 子弹（默认无限，见配置） | 8 | 悬停射击，每5tick发1发 |
| 俯冲轰炸机编队 | 航空炸弹（1枚/次） | 10 | 爬升后俯冲接触投弹 |
| 鱼雷轰炸机编队 | 航空鱼雷（1枚/次） | 12 | 低飞，距目标20-30格投鱼雷 |
| 水平轰炸机编队 | 航空炸弹（8枚） | 12 | 目标上方32格水平投弹 |
| **反潜机编队** | 深水炸弹 | — | 飞至目标上空投放深弹，仅攻击潜艇和水生生物 |
| **侦察机编队** | 航空燃料 | 8 | 切换至侦察机视角，WASD操控 |

#### 9.2 航空弹药与燃料

将弹药放入**弹药库**：

| 弹药 | 用途 |
|------|------|
| 航空燃料 | 变身时自动消耗，为飞机填充燃料；侦察机放飞时消耗 |
| 子弹 | 战斗机专用 |
| 航空炸弹 | 俯冲轰炸机/水平轰炸机 |
| 航空鱼雷 | 鱼雷轰炸机专用 |

> ⚠️ 弹药库中须有**航空燃料**，否则变身时飞机不整备，无法放飞。

**舰装核心燃料**：除航空燃料外，舰装核心也接受以下物品作为燃料：

| 物品 | 说明 |
|------|------|
| 岩浆桶 | 1桶 = 1燃料，返还空桶 |
| 煤炭 | 1个 = 1燃料，支持批量放入 |
| 烈焰棒 | 1个 = 1燃料，支持批量放入 |

**飞行燃油消耗**：飞机在巡航、攻击、侦察中每4tick消耗1燃油。燃油耗尽时飞机**立即返航**，不会在空中熄火坠毁。飞机物品 tooltip 会显示当前燃料加注状态（已加注/未加注）。

#### 9.3 编组 GUI

> 编组功能**默认隐藏**。需要在 `.minecraft/config/piranport-client.toml` 中设置 `flightGroupEnabled = true` 后重启游戏，才会在舰装栏 GUI 中显示"编组"按钮。

**变身状态按 `U`键**（或在舰装栏 GUI 点击"编组"按钮）打开编组界面：

- 共4组（正规航母4组可用）
- **左键**点击飞机槽位：加入/移出当前编组
- **右键**点击已分配槽位：循环切换该槽弹种（---/燃料/航弹/鱼雷/子弹）
- 底部彩色条带显示当前弹种（灰=未配置，琥珀=燃料，红=航弹，蓝=鱼雷，灰白=子弹）
- 攻击模式：**集火**（全组打同一目标）/ **分散**（各自打最近目标）/ **跟随**（跟随侦察机，见9.7）

> ⚠️ 未指定弹种的飞机槽**无法放飞**。

#### 9.4 放飞操作

| 操作 | 效果 |
|------|------|
| 右键（手持核心，变身中） | 按出击顺序放飞下一个未出击编组 |
| 数字键1-4 + 右键 | 放飞指定编号的编组 |
| 右键点击己方飞机实体 | 立即召回，返回武器栏 |

放飞时自动扣除对应弹药。返航后飞机自动回到核心武器栏原槽位。

#### 9.5 火控系统

| 按键 | 功能 |
|------|------|
| `P` | 准星指向生物时锁定/切换目标（范围80格） |
| `O` | 加选目标（追加到锁定列表） |
| `I` | 取消所有锁定 |

锁定目标显示在屏幕右上角 HUD（菱形标记+血量+名称）。

- **集火模式**：所有飞机攻击锁定列表第一个目标
- **分散模式**：每架飞机攻击锁定列表中离自己最近的目标
- **无锁定时**：战斗机优先攻击敌方飞机，其次自动索敌32格内最近敌对生物

**战斗机对空/对地切换（U键）**：

- **默认**：战斗机为**仅对空**模式，只攻击火控锁定的空中目标和自动索敌的敌方飞机
- 按 **U 键**可切换为**对地攻击**模式，允许战斗机攻击地面目标
- **攻击机（轰炸机/鱼雷机）不参与对空作战**，仅战斗机能进行空战

#### 9.6 战场高亮

按 `Y` 键切换**战场高亮模式**：

- 己方飞机、鱼雷、炸弹、子弹发光
- 火控锁定的目标发光
- 仅自己可见，不影响他人

#### 9.7 侦察机与编队跟随

放飞**侦察机编队**后，视角自动切换到侦察机：

- **WASD** 控制侦察机前后左右飞行，**空格/Shift** 控制升降
- **侦察机不能悬停**：无输入时沿当前朝向以最低速度（15%）持续前进
- 侦察机最远可在玩家 200格 范围内飞行，自动加载周围区块
- 按 `G`（或发送退出包）退出侦察视角，返回玩家视角；侦察机继续飞行直到返航
- 侦察机模式下**空手右键被禁用**，防止误触导致皮肤还原或召回舰载机

**编队跟随侦察机**：
1. 在编组 GUI 中将护卫战斗机的攻击模式设为"**跟随**"
2. 放飞侦察机后，设为跟随的战斗机会自动绕侦察机轨道飞行
3. 侦察机移动到哪里，护卫战斗机就跟到哪里
4. 发现敌方目标后，战斗机会立即切换为攻击模式

#### 9.8 空战系统

战斗机在飞行时有血量（20点）：

- 被敌方炮弹/鱼雷/其他伤害击中时扣血
- 血量耗尽：飞机爆炸销毁，掉落物品
- 战斗机子弹命中其他飞机实体时造成伤害，实现空战
- 飞机被击落会在玩家屏幕上显示字幕通知
- **击坠掉落物防爆+浮水**：飞机被击落后掉落的物品不会被爆炸摧毁（类似下界之星），落入水中会自动上浮至水面

#### 9.9 飞机手动补给

飞机出击中，**手持补给物品右键点击己方飞机**可以进行补给（不会触发召回）：

| 手持物品 | 效果 |
|---------|------|
| 航空燃料 | 将该飞机燃油补充至满 |
| 航空炸弹 | 将该飞机炸弹弹药补充至满（适用于俯冲/水平轰炸机） |
| 航空鱼雷 | 将该飞机鱼雷弹药补充至满（适用于鱼雷轰炸机） |

> 手持**空手**右键才会召回飞机，手持物品右键为补给逻辑。

#### 9.10 反潜机（ASW）

反潜机是专门用于攻击水下目标的飞机类型：

| 反潜机 | 弹药容量 | 血量 | 航速 |
|--------|---------|------|------|
| 剑鱼MK3 | 6×深弹 | 4 | 52节 |
| TBF（反潜） | 4×深弹 | 5 | 56节 |

**攻击特性**：
- **仅攻击潜艇和水生生物**（Drowned、Guardian、深海潜艇等），不攻击地面/空中目标
- 飞至目标上空后投放深水炸弹
- 配备**声呐探测**：每20tick扫描周围水下实体，被探测到的目标会显示**黄色高亮描边**
- 声呐高亮独立于火控高亮和Y键战场高亮

**弹药**：使用**深水炸弹**作为弹药（与深弹投射器相同弹种）。

#### 9.13 易燃易爆

变身后若飞机有燃料（即已整备），会获得**易燃易爆**状态效果：
- 受到伤害时额外获得火焰伤害
- 每40tick有15%概率小型爆炸

解除变身后自动移除此效果。

#### 9.14 飞机防护机制

| 情况 | 行为 |
|------|------|
| 滞空超过10分钟 | 强制返航 |
| 距玩家超过48格（跟随/侦察模式为200格） | 传送回玩家附近，切为返航 |
| 连续60tick无移动 | 强制召回（侦察模式不检测） |
| 玩家死亡/下线 | 全部飞机立即召回 |

#### 9.15 战斗机自动升空（舰装配置）

- **GUI模式**：打开**舰装核心 GUI**，在底部"**舰装配置**"区域点击"**战斗机自动升空: [关]**"开启此功能。
- **无GUI模式**：变身后按 **H键** 切换战斗机自动升空的开关。

开启后，系统会同时启用两项自动战斗功能：

**① 战斗机自动放飞**：当以下条件**同时满足**时，每2秒自动放飞一架战斗机：

1. 舰装已变身
2. 64格范围内有**敌对飞行生物**（幻翼/恶魂/恼鬼等）
3. 有配置了弹种的战斗机槽位（未出击或已返航）
4. 弹药库有对应弹药

**触发条件（飞行生物）**：幻翼（Phantom）、恶魂（Ghast）、恼鬼（Vex）以及所有实现飞行行为的敌对生物。

**补给循环**：战斗机弹药耗尽返航后，系统自动用弹药库中的航空燃料补充油料，然后在下次检测时重新出击。若航空燃料或战斗机弹药耗尽，循环暂停直到手动补给。

**② 防空导弹自动发射**：当快捷栏中有防空导弹发射器且32格内有**空中敌对怪物**时，自动消耗背包中的导弹弹药并发射：

- 优先攻击火控锁定的空中目标
- 无锁定时搜索最近的空中敌人
- 排除水下目标和容器实体
- 每个发射器独立冷却，不影响手动发射

> 💡 开启前请确保在编组 GUI 中为战斗机槽位配置了子弹弹种，并在弹药库中备足子弹和航空燃料。防空导弹自动发射需要快捷栏有防空导弹发射器和背包中有对应导弹弹药。

### 11. 副本系统

#### 11.1 进入副本

1. 找到或放置**副本讲台**（DungeonLectern）
2. 右键打开副本书界面，选择章节和关卡
3. 使用对应的**副本钥匙**（通过击败敌人或任务获得）进入

#### 11.2 副本流程

- 副本在独立维度 `piranport:dungeon` 中运行，每组玩家拥有独立实例
- 每个关卡由多个**节点**组成，节点间通过**传送门**连接
- 节点类型：

| 类型 | 说明 |
|------|------|
| 战斗节点 | 击败所有敌人后开放出口传送门 |
| 资源节点 | 提供补给（燃料/图腾等） |
| 消耗节点 | 需要消耗指定物品才能通过 |
| Boss节点 | 最终节点，击败Boss通关 |

#### 11.3 战利品

- 通关节点后会生成**箱子船**（LootShip），右键收集战利品
- 首次通关有额外奖励

#### 11.4 副本道具

| 物品 | 说明 |
|------|------|
| 副本钥匙 | 消耗品，用于开启特定关卡 |
| 回城卷轴 | 使用后弹出确认界面，确认后传送回主世界 |

#### 11.5 脚本化节点

部分节点有特殊脚本流程，例如"炮击引入"脚本：
1. 运输机空投物资
2. 玩家拾取装备
3. 进入战斗阶段

## 常见问题

**Q: 变身后无法使用武器？**
A: 无GUI模式：确认核心在副手，武器在主手/背包中；检查弹药是否足够，负重是否超出核心上限。

**Q: 装甲板放到背包里没有护甲加成？**
A: 无GUI模式下装甲板必须存入核心才生效。打开背包(E)，右键拿起装甲板，再右键点击副手的核心格即可存入。

**Q: 核心放副手后没有自动变身？**
A: 确认 `shipCoreGuiEnabled = false`（默认值）。GUI模式下需手动Shift+右键变身。

**Q: 飞机放不出去？**
A: 检查：①编组GUI是否已通过 `flightGroupEnabled = true` 开启；②编组GUI中是否已为该槽配置弹种；③弹药库是否有对应弹药；④弹药库是否有航空燃料（变身时需整备）。

**Q: 右键飞机没有补给，而是把飞机召回了？**
A: 手持**空手**右键才会召回；手持**航空燃料**或对应弹药物品右键才是补给。确认手中有物品。

**Q: 战斗机弹尽返航，我想让它无限战斗？**
A: 在 `piranport-common.toml` 中设置 `fighterAmmoEnabled = false`（默认即如此），战斗机子弹无限，只消耗燃油。

**Q: 飞机弹尽就返航，能不能自动补给继续战斗？**
A: 在 `piranport-common.toml` 中设置 `autoResupplyEnabled = true`，飞机弹尽时会自动从弹药库抽取对应弹药补充，无需返航。

**Q: 侦察机放飞后看不到画面切换？**
A: 侦察机从 LAUNCHING 状态到 RECON_ACTIVE 需要约 1-2 秒爬升时间，稍等即可。

**Q: 战斗机自动升空不工作？**
A: 检查：①舰装配置中开关是否为"[开]"；②已变身；③编组GUI中战斗机槽已配置弹种；④弹药库有子弹和航空燃料；⑤64格内有敌对飞行生物（幻翼/恶魂/恼鬼等）。

**Q: 防空导弹自动发射不工作？**
A: 检查：①H键自动升空已开启；②快捷栏中有防空导弹发射器；③背包中有对应导弹弹药；④32格内有空中敌对怪物（必须是非地面、非水下的飞行状态敌人）。

**Q: 厨锅不烹饪？**
A: 确认锅正下方有热源（营火需点燃，熔炉需在燃烧状态）。

**Q: 酿造台没有配方？**
A: 水瓶放在底部三槽，试剂放在顶部槽，确保使用原版酿造台。

**Q: 食物放置后不显示？**
A: 渲染基于 BlockEntityRenderer，需要客户端加载完毕。

**Q: 飞机一直不攻击？**
A: 用 P 键锁定目标，飞机才会从巡航状态切换到攻击状态。无锁定时在32格内自动索敌。战斗机默认仅对空，按 U 键切换为对地模式才能攻击地面目标。

**Q: 飞机飞着飞着就自动回来了？**
A: 飞机飞行中会消耗燃油，燃油耗尽后自动返航。确保弹药库有充足的航空燃料。

**Q: 无GUI模式怎么召回所有飞机？**
A: 变身状态下**空手右键**即可召回所有己方飞机。

**Q: 大炮怎么手动装弹？**
A: 主手持大型或中型火炮，按 **R 键**手动装填。系统会自动从背包中寻找匹配弹药。

**Q: 副本钥匙怎么获得？**
A: 通过击败特定敌人或完成任务获得。钥匙绑定特定关卡，在副本讲台使用。

**Q: 怎么制盐？**
A: 点燃熔炉，在熔炉上方放置水源方块，等待10秒蒸发后获得盐块。流动水会产出较少的盐片。

**Q: 导弹怎么锁定目标？**
A: 反舰导弹默认追踪前方最近目标。也可用火控系统（P键）先锁定目标再发射，导弹会追踪锁定目标。

**Q: 反潜机不攻击地面上的怪？**
A: 正常行为。反潜机只攻击潜艇和水生生物（溺尸、守卫者、深海潜艇等），不会攻击地面或空中目标。

**Q: 反潜机的黄色高亮是什么？**
A: 反潜机的声呐探测功能。声呐每秒扫描周围水下实体并以黄色描边标记，帮助你发现水下目标。

**Q: 怎么合成武器？**
A: 使用**武器合成台**（右键打开GUI），选择标签页和具体武器，放入材料后点击合成。部分高级武器还需要蓝图。

**Q: 深海NPC太强打不过？**
A: 可在配置中调低 `abyssalEnemyDifficultyMultiplier`（默认1.0）。建议先探索补给站，装备好后再挑战前哨站和深海基地。注意集群AI——尽量不要同时引起整个舰队的注意。
