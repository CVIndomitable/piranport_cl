## 3. 逐项决策过程

### 3.1 飞机本质定位

**讨论：** 心宿二方案把飞机作为弹药（甲板/弹射器是武器，消耗耐久），受受方案放飞后变生物实体、返航变掉落物，3.1 结论取消放飞器让飞机直接武器化。

**决策：飞机 = 武器栏物品，手持右键放飞，返航自动回武器栏。** 采纳 3.1 结论和总策划案。

### 3.2 整备方式

**讨论：** 总策划案要求在舰装栏 GUI 里拖动燃料/航弹到飞机上，3.1 结论用弹药库顺序消耗，受受方案用不同物品代表不同挂载。

**决策：混合方案。**
- 燃料：自动补充，变身时统一从弹药库扣航空燃料填满所有飞机
- 弹药：由编组 GUI 指定（见 3.6 编组 GUI）
- 变身状态 + 有满油飞机 → 挂易燃易爆 debuff（受伤加成 + 概率爆炸额外伤害）
- 弹药库没有航空燃料时飞机不能放飞

### 3.3 火控按键

**讨论：** 讨论稿结论支持多目标（P/O/I），总策划案为单目标。

**决策：多目标（P 锁定/切换、O 加选、I 取消全部）。**

### 3.4 侦察机

**讨论：** 侦察机视角切换是航空系统中技术难度最高的部分（假人模型 + TP + 区块加载）。4.4 修订已有收敛方案但仍复杂。

**决策：v0.4.0 不做侦察机，只做战斗机和攻击机。** 侦察机留 v0.7.0 Aviation+ 版本。

### 3.5 自动索敌范围

**讨论：** 讨论稿用两层范围（自身 16 格索敌 + 与玩家 80 格控制距离），总策划案统一 32 格。

**决策：MVP 先用 32 格统一范围，后续再分化。**

### 3.6 编队 vs 单机

**讨论：** 2.12 讨论的编队方案是单实体换贴图和名称、调整伤害。

**决策：采用编队方案。** 单实体，模型和名称显示为"飞行编队"。

### 3.7 攻击模式分化

**讨论：** 既然选了多目标火控，AI 不该太简化。

**决策：四种全做。**
- 战斗机：近战咬尾，每秒面板伤害
- 俯冲轰炸机：俯冲接触 + 50%概率着火 30s
- 鱼雷攻击机：贴水面低空投鱼雷（复用 TorpedoEntity）
- 水平轰炸机：高空 32 格水平投弹（新 AerialBombEntity）

## 4. 编组 GUI 设计

项目负责人提出新增编组 GUI，作为航母玩家的"作战指挥台"。

### 4.1 飞机位置

**决策：飞机仍在武器栏，编组 GUI 只做排列分组。** 不把飞机移到专属格子。

### 4.2 编组 GUI 功能

**决策：全部四项功能都做。**
- 分组：哪几架飞机一起出击
- 弹药选择：为编组指定弹种（航弹/航空鱼雷/空战）
- 出击顺序：先放哪组后放哪组
- 攻击模式：集火（全组打同一个目标）/ 分散（各打最近）

### 4.3 弹药选择与弹药库的关系

**决策：编组 GUI 完全取代弹药库顺序（航空线）。** 没在编组 GUI 指定弹种的飞机不能放飞。炮弹和鱼雷仍走弹药库顺序。

### 4.4 编组 GUI 打开方式

**决策：舰装栏 GUI 里加一个"编组"按钮，点击打开。**

### 4.5 变身后能否编辑

**决策：变身前后都能编辑。** 允许航母在战斗中临时调整编组策略。

### 4.6 编组数量

**决策：固定 4 组。** 正航可用 4 组，轻航可用 3 组，由舰装核心决定。

### 4.7 放飞操作

**决策：**
- 直接右键：按出击顺序放飞下一个未出击编组
- 数字键(1-4) + 右键：放飞指定编组

### 4.8 飞行中飞机的编组 GUI 显示

**决策：飞行中的飞机在编组 GUI 中变灰不可编辑，返航后恢复。**

## 5. 异常处理与防御机制

### 5.1 飞机异常防御

**讨论：** 4.4 修订中已提到飞出加载区块问题，"飞狗方案"（套用狼传送代码）因寻路难实现未被采纳。

**决策：四重防御全部启用。**

| 机制 | 说明 |
|------|------|
| 滞空时间硬上限 | 10 分钟（12000 tick），到时强制回收 |
| 距离上限 + TP | 超过 48 格传送回玩家附近，切为返航 |
| 卡死检测 | 连续 3 秒位置变化 < 0.1 格，强制回收 |
| 空手右键 | 玩家空手右键自己的飞机可立即回收 |

### 5.2 玩家死亡

**决策：所有在飞的飞机立即回收到武器栏。** 安全但不惩罚。

### 5.3 玩家下线/断线

**决策：所有在飞的飞机安静回收到武器栏。**

## 6. 数值与测试工具

### 6.1 滞空时间

**决策：硬上限 10 分钟。** 不论飞机燃料容量多大都不超过此值。

### 6.2 浮动靶子

**新增测试工具：** `FloatingTargetEntity`（浮动靶子）

- 模型：复用原版盔甲架
- 功能：可穿原版盔甲，获得盔甲效果
- 物理：放置在水面上漂浮
- 用途：伤害测试靶，验证四种攻击模式的伤害数值和护甲减伤
- 放置：手持靶子物品右键水面
