# 皮兰港模组教程

本教程面向玩家，介绍皮兰港模组的核心玩法。

---

## 模组功能概览

### 1. 舰装系统

舰娘核心机制。手持**舰装核心**（小/中/大型）Shift+右键变身：

- 变身后进入舰装模式，右键打开舰装栏 GUI
- **武器栏**（左侧）：装备火炮或鱼雷发射器
- **弹药库**（中间）：存放对应炮弹/鱼雷
- **强化栏**（右侧）：装备附加装甲（+2/+4/+6防御）
- **负重系统**：武器+弹药+装甲总重量不超过核心上限，超重无法装备
- **移速**：负重越接近上限，移速越慢（最低约40%）

#### 武器切换
`Tab`（默认）切换当前选中武器槽，右键发射当前武器。

### 2. 火炮战斗

| 武器 | 弹药 | 特性 |
|------|------|------|
| 小型火炮 | 小口径 HE/AP | 装填快，伤害低 |
| 中型火炮 | 中口径 HE/AP | 均衡 |
| 大型火炮 | 大口径 HE/AP | 伤害高，装填慢 |

- **HE炮弹**：爆炸伤害，范围溅射
- **AP炮弹**：穿甲，对高护甲目标更有效

### 3. 鱼雷系统

| 物品 | 说明 |
|------|------|
| 533mm标准鱼雷 | 基础鱼雷弹药 |
| 610mm氧气鱼雷 | 高伤害鱼雷弹药 |
| 双联装鱼雷发射器 | 2枚，±3°散布 |
| 三联装鱼雷发射器 | 3枚，±4°散布 |
| 四联装鱼雷发射器 | 4枚，±6°/±2°散布 |

- 鱼雷在水面贴合航行，命中目标施加**进水**Debuff（每秒魔法伤害）
- 飞行中遇水自动进入水面模式

1. 手持原料右键放上去
2. 空手右键切割（每次+1进度）
3. 达到所需刀数后自动弹出产物
4. Shift+右键取回未切完的食材

## 常见问题

**Q: 变身后无法使用武器？**
A: 检查弹药库是否有对应弹药，负重是否超出核心上限。

**Q: 厨锅不烹饪？**
A: 确认锅正下方有热源（营火需点燃，熔炉需在燃烧状态）。

**Q: 酿造台没有配方？**
A: 水瓶放在底部三槽，试剂放在顶部槽，确保使用原版酿造台。

**Q: 食物放置后不显示？**
A: 渲染基于 BlockEntityRenderer，需要客户端加载完毕。
