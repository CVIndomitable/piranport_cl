# 皮兰港模组教程

本教程面向玩家，介绍皮兰港模组的核心玩法。

---

## 模组功能概览

### 1. 舰装系统

舰娘核心机制。手持**舰装核心**（小/中/大型）Shift+右键变身：

- 变身后进入舰装模式，右键打开舰装栏 GUI
- **武器栏**（左侧）：装备火炮、鱼雷发射器或飞机编队
- **弹药库**（中间）：存放炮弹/鱼雷/航空弹药/航空燃料
- **强化栏**（右侧）：装备附加装甲（+2/+4/+6防御）
- **负重系统**：武器+装甲总重量不超过核心上限，超重无法装备
- **移速**：负重越接近上限，移速越慢（最低约40%）

#### 武器切换

`V`键切换当前选中武器槽，右键发射/放飞当前武器。

#### 装填进度显示

变身后，**快捷栏中舰装核心图标**上会显示彩色装填进度条（每个武器槽一条）：

| 颜色 | 武器类型 | 说明 |
|------|---------|------|
| 橙红色 | 火炮 | 亮橙 = 可发射 |
| 蓝色 | 鱼雷发射器 | 亮蓝 = 可发射 |
| 绿色 | 飞机编队 | 亮绿 = 可放飞 |

条从左向右填满表示装填完成，变亮即可再次发射。

### 2. 火炮战斗

| 武器 | 弹药 | 特性 |
|------|------|------|
| 小型火炮 | 小口径 HE/AP | 装填快，伤害低 |
| 中型火炮 | 中口径 HE/AP | 均衡 |
| 大型火炮 | 大口径 HE/AP | 伤害高，装填慢 |

- **HE炮弹**：爆炸伤害，范围溅射
- **AP炮弹**：穿甲，对高护甲目标更有效

### 3. 鱼雷系统

| 物品 | 说明 |
|------|------|
| 533mm标准鱼雷 | 基础鱼雷弹药 |
| 610mm氧气鱼雷 | 高伤害鱼雷弹药 |
| 双联装鱼雷发射器 | 2枚，±3°散布 |
| 三联装鱼雷发射器 | 3枚，±4°散布 |
| 四联装鱼雷发射器 | 4枚，±6°/±2°散布 |

- 鱼雷在水面贴合航行，命中目标施加**进水**Debuff（每秒魔法伤害）
- 飞行中遇水自动进入水面模式

| 原料 | 产物 |

1. 手持原料右键放上去
2. 空手右键切割（每次+1进度）
3. 达到所需刀数后自动弹出产物
4. Shift+右键取回未切完的食材

### 5. 航空系统

#### 5.1 飞机编队物品

四种飞机编队放入武器栏，变身后通过编组配置后右键放飞：

| 飞机 | 弹药 | 伤害 | 弹药容量 | 负重 | 攻击方式 |
|------|------|------|---------|------|---------|
| 战斗机编队 | 子弹 | 18（每发18/8≈2.25） | 64发 | 12 | 悬停射击，每5tick发1发 |
| 俯冲轰炸机编队 | 航空炸弹 | 24（×1.5=36） | 1枚/次 | 16 | 爬升后俯冲接触投弹 |
| 鱼雷轰炸机编队 | 航空鱼雷 | 30 | 1枚/次 | 20 | 低飞，距目标20-30格投鱼雷 |
| 水平轰炸机编队 | 航空炸弹 | 36（×1.5=54） | 8枚 | 24 | 目标上方32格水平投弹 |

#### 5.2 航空弹药

将弹药放入**弹药库**：

| 弹药 | 用途 |
|------|------|
| 航空燃料 | 变身时自动消耗，为飞机填充燃料 |
| 子弹 | 战斗机专用 |
| 航空炸弹 | 俯冲轰炸机/水平轰炸机 |
| 航空鱼雷 | 鱼雷轰炸机专用 |

> ⚠️ 弹药库中须有**航空燃料**，否则变身时飞机不整备，无法放飞。

#### 5.3 编组 GUI

**变身状态按 `U`键**（或在舰装栏 GUI 点击"编组"按钮）打开编组界面：

- 共4组（正规航母4组可用）
- **左键**点击飞机槽位：加入/移出当前编组
- **右键**点击已分配槽位：循环切换该槽弹种（---/燃料/航弹/鱼雷/子弹）
- 底部彩色条带显示当前弹种（灰=未配置，琥珀=燃料，红=航弹，蓝=鱼雷，灰白=子弹）
- 攻击模式：**集火**（全组打同一目标）/ **分散**（各自打最近目标）

> ⚠️ 未指定弹种的飞机槽**无法放飞**。

#### 5.4 放飞操作

| 操作 | 效果 |
|------|------|
| 右键（手持核心，变身中） | 按出击顺序放飞下一个未出击编组 |
| 数字键1-4 + 右键 | 放飞指定编号的编组 |
| 右键点击己方飞机实体 | 立即召回，返回武器栏 |

放飞时自动扣除对应弹药。返航后飞机自动回到核心武器栏原槽位。

#### 5.5 火控系统

| 按键 | 功能 |
|------|------|
| `P` | 准星指向生物时锁定/切换目标（范围80格） |
| `O` | 加选目标（追加到锁定列表） |
| `I` | 取消所有锁定 |

锁定目标显示在屏幕右上角 HUD（菱形标记+血量+名称）。

- **集火模式**：所有飞机攻击锁定列表第一个目标
- **分散模式**：每架飞机攻击锁定列表中离自己最近的目标
- **无锁定时**：自动索敌32格内最近敌对生物

#### 5.6 战场高亮

按 `Y` 键切换**战场高亮模式**：

- 己方飞机、鱼雷、炸弹、子弹发光
- 火控锁定的目标发光
- 仅自己可见，不影响他人

#### 5.7 易燃易爆

变身后若飞机有燃料（即已整备），会获得**易燃易爆**状态效果：
- 受到伤害时额外获得火焰伤害
- 每40tick有15%概率小型爆炸

解除变身后自动移除此效果。

#### 5.8 飞机防护机制

| 情况 | 行为 |
|------|------|
| 滞空超过10分钟 | 强制返航 |
| 距玩家超过48格 | 传送回玩家附近，切为返航 |
| 连续60tick无移动 | 强制召回 |
| 玩家死亡/下线 | 全部飞机立即召回 |

## 常见问题

**Q: 变身后无法使用武器？**
A: 检查弹药库是否有对应弹药，负重是否超出核心上限。

**Q: 飞机放不出去？**
A: 检查：①编组GUI中是否已为该槽配置弹种；②弹药库是否有对应弹药；③弹药库是否有航空燃料（变身时需整备）。

**Q: 厨锅不烹饪？**
A: 确认锅正下方有热源（营火需点燃，熔炉需在燃烧状态）。

**Q: 酿造台没有配方？**
A: 水瓶放在底部三槽，试剂放在顶部槽，确保使用原版酿造台。

**Q: 食物放置后不显示？**
A: 渲染基于 BlockEntityRenderer，需要客户端加载完毕。

**Q: 飞机一直不攻击？**
A: 用 P 键锁定目标，飞机才会从巡航状态切换到攻击状态。无锁定时在32格内自动索敌。
